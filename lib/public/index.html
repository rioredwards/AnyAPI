Animals and Plants

This API uses lists of plants and animals as its resources. The routes are as follows:
Route Method Description Example 
/animals
 GET Gets a list of animals' names and ids List Animals 
/animals/:id
 GET Returns the detail about the animal matching the id in the parameter Get Cat 1 
/plants
 GET Gets a list of plants' names and ids List Plants 
/plants/:id
 GET Returns the detail about the plant matching the id in the parameter Get Plant 1
